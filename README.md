FriendMatch 是一个基于兴趣和性格匹配的交友平台，帮助用户找到志趣相投的朋友。用户可以通过填写个人资料、兴趣标签、和个性测试来找到合适的匹配对象。该项目主要针对那些希望扩大社交圈、寻找共同爱好的用户群体
